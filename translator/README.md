# llm_translator_srt
通过大模型将英文字幕翻译成中文字幕



# start
1. create .env in colab.com 
```
OPEN
INPUT_FILE
OUTPUT_FILE
```
3. run


# Other Tools 

# OPENAI
OPENAI_API_KEY=
OPENAI_API_BASE=

# SRT FILE
INPUT_PATH=<输入srt文件位置>
OUTPU_TPATH=<输入翻译后srt文件位置>
MAXLINE=<每次处理多少行>
